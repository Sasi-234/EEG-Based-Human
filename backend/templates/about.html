About the Project

Overview

The EEG-Based Human Emotion Recognition System is an advanced AI-powered application designed to detect and analyze human emotions using electroencephalogram (EEG) brainwave signals.

Technology Stack

Frontend
  • HTML5, CSS3, JavaScript
  • Bootstrap 5
  • Chart.js for visualizations
  • jQuery
Backend
  • Python 3.8+
  • Django 4.2
  • Django REST Framework
  • SQLite Database
AI/ML
  • TensorFlow 2.15
  • Keras
  • CNN & LSTM Models
  • NumPy, Pandas, Scikit-learn
  • MNE-Python for EEG processing
Dataset
  • DEAP Dataset
  • 32 EEG channels
  • 6 emotion categories
  • Valence & Arousal metrics

Detected Emotions

Our system can accurately detect the following six emotion categories:

Happy

High valence, high arousal

Sad

Low valence, low arousal

Angry

Low valence, high arousal

Relaxed

High valence, low arousal

Stressed

Low-mid valence, high arousal

Excited

High valence, very high arousal

Applications

  • Healthcare: Mental health monitoring and diagnosis
  • Human-Computer Interaction: Adaptive interfaces based on user emotions
  • Research: Emotion analysis and psychological studies
  • Wellness: Stress management and emotional well-being tracking

How It Works

  1. Data Upload: Users upload EEG signal files in supported formats
  2. Preprocessing: Signals are filtered, normalized, and cleaned
  3. Feature Extraction: Relevant features are extracted from the signals
  4. AI Analysis: Deep learning models (CNN/LSTM) analyze the patterns
  5. Emotion Detection: System predicts the emotion with confidence score
  6. Recommendations: Personalized suggestions based on detected emotions
Note

This system is designed for research and educational purposes. For clinical applications, please consult with healthcare professionals.
